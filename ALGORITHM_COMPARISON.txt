单步解码 vs 多步解码：对比说明
═══════════════════════════════════════════════════════════════════════════════

【单步解码】decode_by_bert() - reader.py
─────────────────────────────────────────────────────────────────────────────

初始输入: [CLS] (MASK) sent1 (MASK) sent2 (MASK) sent3 (MASK) sent4 (MASK) sent5 [SEP]
                ^pos1   ^pos2   ^pos3   ^pos4   ^pos5

步骤1（唯一步骤）:
  ├─ 获取所有5个MASK位置的logits
  ├─ 对每个位置独立进行argmax
  └─ 返回: [label1, label2, label3, label4, label5]

特点:
  ✓ 快速（单次前向传递）
  ✗ 无上下文依赖（位置间独立）
  ✗ 可能有重复标签


【多步解码】multi_step_decode() - multi_step_decode.py
─────────────────────────────────────────────────────────────────────────────

初始输入: [CLS] (MASK) sent1 (MASK) sent2 (MASK) sent3 (MASK) sent4 (MASK) sent5 [SEP]
                ^1    ^2    ^3    ^4    ^5

第一步（选最高置信度位置）:
  输入: 5个MASK -> 获取logits -> 选置信度最高的位置（假设位置3）
  │
  ├─ logit[pos1].max = 0.45
  ├─ logit[pos2].max = 0.38
  ├─ logit[pos3].max = 0.72  ◄─── 最高 ★
  ├─ logit[pos4].max = 0.51
  └─ logit[pos5].max = 0.41
  
  动作: pos3 -> token_id_for_label_3 -> 替换MASK
  结果: [CLS] (MASK) sent1 (MASK) sent2 [token3] sent3 (MASK) sent4 (MASK) sent5 [SEP]
                ^1    ^2    ^3    ^4    ^5
                      (不再是MASK了)

第二步（4个MASK剩余）:
  输入: 4个MASK -> 获取logits -> 选最高置信度（假设位置1）
  
  logit[pos1].max = 0.68  ◄─── 最高 ★
  logit[pos2].max = 0.55
  logit[pos4].max = 0.42
  logit[pos5].max = 0.39
  
  动作: pos1 -> token_id_for_label_1 -> 替换MASK
  结果: [CLS] [token1] sent1 (MASK) sent2 [token3] sent3 (MASK) sent4 (MASK) sent5 [SEP]

... 继续直到没有MASK ...

第五步（最后1个MASK）:
  输入: 1个MASK -> 获取logits -> 预测该位置
  
  结果: 全部位置都被填充，返回 [label1, label3, label2, label4, label5]

特点:
  ✓ 后续预测有上下文（前面位置已填充）
  ✓ 优先预测高置信度位置（渐进优化）
  ✓ 避免重复标签（每个位置只预测一次）
  ✗ 计算量大（5次前向传递）

算法对比总结
═══════════════════════════════════════════════════════════════════════════════

维度              │ 单步解码              │ 多步解码
─────────────────┼────────────────────┼───────────────────────
前向传递次数      │ 1                  │ 5
计算速度          │ 快 ⚡               │ 慢 🐢
上下文利用        │ 无 ❌               │ 有 ✅
重复标签概率      │ 高 ⚠️               │ 低/无 ✅
预测质量          │ 中等                │ 可能更好
自回归性          │ 无                  │ 是
推荐场景          │ 快速评估            │ 高精度需求
